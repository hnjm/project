Idee

Machine Learning stellt einen Aspekt der künstlichen Intelligenz dar, der in der vergangenen Zeit an immer größerer Bedeutung gewonnen hat. In diesem Kontext ist insbesondere das „Deep Learning“ hevorzuheben, das wiederum einen Teilbereich des Machine Learnings darstellt. Dessen Popularität lässt sich zum Einen damit erklären, dass es die Geschwindigkeit und Reife heutiger Prozessoren (CPU/GPU/TPU\footnote{Wikipedia, Tensor Processing Unit. (https://de.wikipedia.org/wiki/Tensor_Processing_Unit)}/FPGA\footnote{Wikipedia, Field Programmable Gate Array. (https://de.wikipedia.org/wiki/Field_Programmable_Gate_Array)}) zulässt Ergebnisse in akzeptabler Zeit zu erzielen und zum Anderen damit, dass durch das stetige Anwachsen der durchs Internet erzeugten Daten, genug Material zur Verfügung steht, mit dem gearbeitet werden kann. Besonders im Kontext von Bildbearbeitungen und –Klassifizierungen sind Techniken des Machine Learnings kaum noch wegzudenken.

Die Idee dieses Projekts besteht darin mit Hilfe des Machine Learnings Bilder, die zuvor unkenntlich gemacht worden sind, wieder herzustellen. Dabei gibt es verschiedene Technologien, die zum Erreichen der Unkenntlichkeit verwendet werden können: zu nennen wären dabei beispielsweise „mosaic“, „pixelation“ und das „blurring“. Durch trainieren eines Systems mit Originalbildern und dem unkenntlichen Pendant dazu, soll das System die Fähigkeit erhalten, aus zuvor unkenntlich gemachten Bildern, die dem System unbekannt sind, Bilder zu generieren, die dem Originalbild möglichst nahe kommen.

Als Ausgangspunkt soll dabei eine Gesichtsdatenbank verwendet werden, die einen großen Bestand an verschiedenen Bildern von Gesichtern darstellt. Mittels entsprechender PhotoShop-Methoden werden die korrespondierenden unkenntlichen Bilder erstellt. Diese Menge an Daten kann dann verwendet werden, um ein Training des Systems vorzunehmen.
